# 全栈博客开发课程大纲

> 20 节课精通现代全栈应用开发 - 从零到部署

## 📋 课程概述

**课程目标：** 通过学习一个真实的全栈博客项目，掌握现代 Web 开发的核心技术栈和最佳实践。

**技术栈：**
- 前端：Nuxt 4 + Vue 3 + TypeScript + Tailwind CSS
- 后端：Express + Prisma + PostgreSQL
- 工程化：Monorepo (pnpm workspace)
- 部署：Vercel + Railway + Supabase

**课程时长：** 20 节课，每节课 1.5-2 小时

**适合人群：**
- 有 HTML/CSS/JavaScript 基础
- 了解 Vue 基础语法
- 想学习全栈开发的前端开发者
- 想了解现代前端架构的后端开发者

---

## 🎯 课程结构

```
第一模块：基础准备（第 1-2 课）
    ↓
第二模块：前端基础（第 3-7 课）
    ↓
第三模块：后端基础（第 8-12 课）

第五模块：进阶优化（第 17-19 课）

```
